DEMO
===========================

###########环境依赖
autoJs 4.1.1+
autoJs Pro v7.0+
坚果云网盘

###########部署步骤
1. 注册坚果云网盘
    地址：https://www.jianguoyun.com

2. 安装相关vscode插件

3. 获取坚果云相关数据
    账号
    密钥
    js文件存放路径


###########目录结构描述
├── 使用说明.md                   // help
├── mod                         // js模块
├── pages                       // UI布局路径
│   ├── home.js                 // 首页布局
│   └── setTing.js              // 设置布局
│── script                      // 脚本文件路径
│    ├── mainActivity.js         // 云端界面生成入口
│    └── script.js               // 主要脚本文件
│── main.js                      // 客户端入口
└── pages.json                   // 模板主要参数配置
